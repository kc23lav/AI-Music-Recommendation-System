# AI Music Recommendation System 

An AI-powered music recommendation system built using Python and Flask that recommends songs based on user preferences and dataset analysis.

---

## Features

- Music recommendation engine
- Dataset-based recommendation logic
- Flask backend integration
- Dynamic recommendation generation
- Modular project architecture

---

## Tech Stack

- Python
- Flask
- Pandas
- Machine Learning Concepts
- HTML/CSS/JavaScript

---

## Project Structure

```bash
backend/
│── app.py
│── recommender.py
│── generate_dataset.py
│── dataset.csv
│── requirements.txt

frontend/

screenshots/
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/kc23lav/AI-Music-Recommendation-System.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
